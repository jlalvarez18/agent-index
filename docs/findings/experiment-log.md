# Agent Index Experiment Log

This log preserves the test-and-benchmark progression for the Graphify v8 prototype. It is intentionally closer to a lab notebook than a polished narrative.

## How To Read This

- `Regression test` names the focused automated test or benchmark check added to catch the behavior.
- `Before` and `After` use the Graphify source-only benchmark when available.
- Split metrics use symbol-first scoring unless the row explicitly says blended.
- A `-` means the metric was not yet available or was not the point of that step.

## Current Conclusion

The three-corpus experiment supports a narrower claim than "symbol search wins": FTS is still the recall backbone, while symbol structure, query intent, and small coding-domain priors improve ranking and navigability. Graphify is saturated, HTTPX shows the soft hybrid can recover exact symbols, and Click shows the next weakness is exact symbol ordering inside already-correct files and neighborhoods. Intent rules need precise trigger conditions; broad natural-language shortcuts can help one question while quietly hurting another.
